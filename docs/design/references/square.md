# Design System

## Overview
Design tokens extracted from squareup.com.

## Colors
- **Primary** (#ffffff): CTAs, active states, key interactive elements
- **Secondary** (#000000): Supporting UI, secondary actions
- **Surface** (#bbbbbb): Page backgrounds
- **On-surface** (#006aff): Primary text

## Typography
- **Headlines**: Exact Block, regular, 40–90px
- **Body**: Cash Sans, medium, 12–74px
- **Labels**: Square Sans Display VF, regular, 10–64px
- **Labels**: Square Sans Text VF, bold, 8.5–24px
- **Labels**: Square Market, bold, 12–24px
- **Labels**: Cash Sans Mono, medium, 14px

## Components
- **Buttons**: Rounded (6px), primary uses #000000 fill, outlined variant available
- **Inputs**: 0px none border, 10px radius
- **Border radius scale**: 6px, 4px, 10px, 50px
- **Elevation**: uses box shadows for depth

## Do's and Don'ts
- Do use rounded corners consistently across interactive elements
- Do use the primary color sparingly — only for the most important action per screen
- Do maintain 4.5:1 contrast ratio for all body text (WCAG AA)
